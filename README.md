## 專題摘要
  近年來,人工智慧生成技術快速發展,Deepfake 影像已能透過臉部替換、表情遷移、嘴型同步與影像合成等方式產生高度逼真的人臉內容。這類影像若被用於詐騙、 假訊息散布或冒名影片,可能造成使用者對網路影像可信度的判斷困難。傳統偵測方式多半需要使用者將圖片或影片另外上傳至網站或軟體進行分析,缺乏在瀏覽過程中即時提醒的能力。←
  本專題以「DeepFake 人臉偵測的應用程式」為主題,規劃開發一套可於Windows 10/11 環境執行的背景偵測工具。系統啟動後會以螢幕擷取模組取得目前畫面,再由人臉偵測模組判斷畫面中是否存在人臉;若偵測到人臉,系統會擷取人臉區域並送入 FakeFormer 深度偽造偵測模型進行真偽判斷。當模型判定畫面中疑似出現Deepfake 偽造人臉時,系統會以被動式警示視窗提醒使用者,以降低使用者在瀏覽網頁、影片或社群內容時誤信偽造影像的風險。←
  本專題重點不僅在於模型判斷結果,也包含背景監控流程、觸發機制、系統效能與使用者互動設計。為避免不必要的運算負擔,系統不直接對所有截圖進行深度偽造判斷,而是先以人臉偵測作為觸發條件;當畫面中沒有人臉時即回到等待與擷取流程, 僅在有臉部區域時才啟動真偽分析。此設計可提升系統即時性,並使警示功能更符合實際使用情境。←
